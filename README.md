##User Stories

As a user, I want to view a list of all HDB resale transactions so I can see market trends.

As a user, I want to click on a specific transaction to see details like the street name, floor area, and remaining lease.

As a user, I want to add a "newly watched" HDB listing or record via a form to track potential purchases.

As a user, I want to see a dashboard that summarizes the total number of listings I'm tracking.

- **Search:** As a user, I want to search for HDB resale prices by town with the remaining lease of the flat
- **Filter:** As a user, I want to filter results by flat type and floor area.
- **View:** As a user, I want to see the trend of sales price over the years
